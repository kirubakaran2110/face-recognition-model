# Dataset Information

This folder should contain the dataset used for training and evaluation.

Please download the VGGFace2 subset manually (e.g., "VGGFace2_subset_500" from Kaggle)
and place it here.

Expected structure:
data/
└── vggface2_subset_500/
    ├── train/
    │   ├── person_0001/
    │   ├── person_0002/
    │   └── ...
    └── val/
        ├── person_0001/
        ├── person_0002/
        └── ...

Download links:
- Official: https://www.robots.ox.ac.uk/~vgg/data/vgg_face2/
- Kaggle subset: https://www.kaggle.com/datasets/rajaychakravarty/vggface2-subset-500
